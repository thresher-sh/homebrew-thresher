# Homebrew Tap for Thresher

AI-powered supply chain security scanner for open source packages.

## Install

```bash
brew tap thresher-sh/thresher
brew install thresher
```

## Usage

```bash
# Build the scanner VM (first time only)
thresher build

# Scan a repository
thresher scan <repo_url>
```
